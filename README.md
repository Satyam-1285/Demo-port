# Demo-port# 🛠️ Engineering Portfolio Webpage

A sleek, responsive portfolio landing page designed specifically for showcasing Electronics and Communication Engineering (ECE) projects and technical skills.

## 🚀 Live Demo
This project is optimized for **GitHub Pages**. Once deployed, it provides a clean, professional dark-mode interface to display technical expertise.

## ✨ Features
- **Dark Aesthetic:** Professional UI designed for engineers and developers.
- **Skill Badges:** Highlights core competencies like Python, IoT, and Arduino.
- **Project Grid:** A clean layout to showcase hardware and software integration projects.
- **Fully Responsive:** Works perfectly on desktops, tablets, and mobile devices.

## 🛠️ Built With
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling with CSS variables and Flexbox/Grid.
- **Glassmorphism:** Subtle visual effects for a modern look.
